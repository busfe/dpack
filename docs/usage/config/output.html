output

output

output默认目录为./dist，静态文件会被打包进./dist/static/目录中，静态文件编译目录位置暂不支持修改，可以通过-d参数修改编译后的目录位置，对于不同的环境我们可能需要加入不同的domain,可以通过以下代码实现。

    if (args.env === 'production') {
    dpack.set('output.publicPath', '/mis/')
  }

env为-e中传递的值，默认dpack server的env为development, dpack build的env为production
