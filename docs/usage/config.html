配置参数

配置参数

dpack默认拥有一套配置规则，推荐基于dpack模板进行开发。当然你可以自定义配置，在项目根目录下新建一个dpack.config.js，如果需要用其他名字，请使用-c指定具体名。配置文件需要export一个function,第一个参数为dpack，通过dpack.set

    module.export = function(dpack, args) {

    }

dpack提供了两个方法：

1、dpack.set

自定义webpack配置

    usage:
    1、dpack.set(obj)
    2、dpack.set(key, value) // key可以用a.b.c表示，比如 dpack.set('output.publicPath', '/mis/')

2、dpack.unset

如果不想用我们提供的webpack配置，你可以通过unset清空配置

    usage:
    dpack.unset(`output.publicPath`)
