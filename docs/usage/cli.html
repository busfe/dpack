命令行

命令行

可以通过dpack -h查看dpack提供的所有命令用法

    ➜  React-Demo dpack -h

      Usage: dpack <command> [options]


      Options:

        -v --version        output the version number
        -w --watch          watch the file change
        -p --port <n>       server port
        -o --optimize       optimize css or js files
        -e --env <s>        node env
        -c --config <path>  custom config path, default is dpack.config.js
        -m --md5            build bundle with hash
        -d --dest <path>    output path
        -l --lanch          lanch browser
        -h, --help          output usage information


      Commands:

        server                            start webpack dev server
        build                             build the project
        create <project-name> <template>  init the project

create

初始化一个项目，<project-name>为项目名，template为模板类型(reactorvue)。

server

启动本地服务进行本地调试，本地调试的时候默认开启热加载模式，如果不传-e，默认加入development环境变量，开发者可以根据process.env.NODE_ENV === 'development' 来判断是否在开发环境中。

build

发布到生产环境，默认发布目录为./dist，开发者可以根据-d去自由定义发布目录，并且默认加入production环境变量。

注意点：在React项目中，需要在build的时候加入-o，对JS&CSS进行压缩，并且进行tree shaking，否则React会提示warning。
