Vue

Vue

初始化一个vue项目

dpack create <project-name> vue

cd <project-name>

dpack server

这样就dpack就会下载vue模板初始化了一个vue项目。

这样，初始化后的vue项目目录如下所示： 

<project-name>/                          // root directory
├── build/                               // vue-loader config directory
│     ├── utils.js                       // css-loader configuration for vue-loader
│     └── vue-loader.conf.js             // vue-loader config
├── src/                                 // src directory
│     ├── assets/                        // static assets directory
│     ├── components/                    // common components of project
│     ├── router/                        // project route config
│     ├── store/                         // vuex config directory
│     ├── routes/                        // bussiness logic pages directory
│     ├──.babelrc                        // babel-loader config file
│     ├── App.vue                        // vue entry
│     └── main.js                        // webpack entry
│     └── layout.html                    // html-webpack-plugin template entry
├── .editorconfig                        // uniform the text editor behavior
├── .eslintignore                        // which paths should be omitted from linting
├── .eslintrc.js                         // specific rules for eslint
├── .gitignore                           // ignore the specific files when using git
├── package.json                         // npm entry file
├── README.md                            // just what you see now

vue模板注意事项

  • dpack初始化的vue项目是基于vue@2.4版本，不支持vue@1.x的项目

  • dpack初始化的vue项目会自带vue-router和vuex

  • 由于vue-loader默认用babel-loader编译vue模板中是script，其推荐使用.babelrc文件方式来配置babel-loader的presets和plugins，所以dpack的vue项目必须配置要有该文件
